# 📊 Data Analytics Portfolio

Welcome to my data analytics portfolio! This repository showcases my projects in **data analysis, visualization, and business intelligence** using Power BI, SQL, and Python.

---

## 🗂️ Projects

### 🚴 [Cyclistic Bike-Share Analysis](./cyclistic-bike-share-analysis)
> Google Data Analytics Capstone Project

Analyzed **6 million rides** to uncover behavioral differences between casual riders and annual members, with actionable marketing recommendations.

- **Tools:** Power BI, Power Query, DAX
- **Key Skills:** Data cleaning, trend analysis, data visualization
- **Highlights:** Monthly trends, ride duration buckets, time-of-day patterns

---

### 🛒 [Superstore Sales Dashboard](./superstore-sales-dashboard)
> Sales & Profit Performance Analysis

Interactive dashboard analyzing sales, profit, and order trends across categories and sub-categories with Year-over-Year growth tracking.

- **Tools:** Power BI, DAX
- **Key Skills:** KPI tracking, profitability analysis, YoY calculations
- **Highlights:** $2.3M sales analyzed, identified loss-making sub-categories

---

### 🏠 [IoT Smart Home Analysis](./iot-smart-home-analysis)
> Real-Time Smart Home Monitoring Dashboard

Monitoring dashboard for power consumption, temperature, humidity, and device status across rooms using a custom 3D map visual.

- **Tools:** Power BI, Custom Map Visual, Power Query
- **Key Skills:** IoT data analysis, energy monitoring, custom visuals
- **Highlights:** Room-level power analysis, device status tracking
